# CLAUDE.md — reelcast

このファイルは Claude Code が reelcast で作業する際の前提・規約。詳細な意思決定の記録は `docs/PLAN.md` を参照（こちらが単一の真実）。

## プロジェクト概要
- 個人開発。YouTube / Instagram 向けに、分析 → マーケ企画 → 制作 → 投稿 → 運用 を AI で一気通貫に行うシステム。
- **土台＝AI機能性音楽（作業用/睡眠/lo-fi/環境音）× 完全フェイスレス × YouTube広告収益**。
- 将来は各工程をマルチエージェントでオーケストレーション。

## 常に効く制約（判断時の前提）
- **最優先は収益化**。施策はこの観点で評価する。
- **作業者の投下時間は〜5時間/週**。だから**自動化を最優先**し、手作業を増やす提案は避ける。
- **予算は無料中心（〜数百円）。唯一の例外は音楽生成ツール（商用権確保のため有料）**。他工程で有料前提の提案をする際は必ず明示し、無料/低コスト代替を併記する。
- **実行環境はローカル Mac 中心**。重い生成・合成はローカルで完結させる。
- **公開は必ず人間承認ゲートを通す**。完全自動公開は採らない。

## 確定したツール（詳細・根拠は docs/TOOLS.md）
- **音楽：Suno Pro（$10/月）＋人間アシスト**。公式APIなしのため、人間がWebで生成・DLし `data/assets/music/<id>/` に投入 → パイプラインが取り込み・検証。**第三者Suno APIラッパーは規約リスクで使わない**。
- **画像：保留**（固定キャラの画風と一緒に決定）。候補：ローカルStable Diffusion / Leonardo無料枠 / Adobe Firefly。いずれも商用可を確認してから。
- ⚠️ 純AI生成楽曲は米国で著作権登録不可 → 自AI音楽のContent ID登録はしない（収益化はプラットフォームの商用ライセンスで可能）。

## 技術スタック
- 言語：**Python**
- データストア：**SQLite（状態管理）＋ ローカルファイル（素材）**
- 映像合成：**コード自動化（ffmpeg / MoviePy 中心）**
- 構成：**工程単位の疎結合モジュール**（企画/音楽/映像/合成/メタ/公開）→ 将来エージェント化
- 設計原則：各工程は**冪等・再実行可能**に。生成は失敗・やり直しが前提。

## コンテンツ仕様
- 長尺：テーマ別シリーズ（45分〜1時間）。YouTube が収益本体。
- ショート：長尺の切り出し。Instagram は認知・集客（当面 手動/半自動）。
- 世界観：**まったり猫マスコット**（インスト機能性音楽の看板。歌わない）。キャラは**固定アセットとして一度作り、全動画で使い回す**（毎回 AI 生成しない＝一貫性が崩れるため）。ブランド軸＝猫は不変、シリーズ＝世界（Neon Night / Cozy Cabin / Starship）で可変。詳細は docs/CONCEPT.md。
- ⚠️ 既存IP（ガンダム等のメカ、既存の猫lofiチャンネル名）は回避し、**完全オリジナル**で作る。
- 頻度：長尺 週2 ＋ ショート 週3〜4。

## ⚠️ 必ず守るルール（規約・権利）
- **規約フル遵守**。「バレなければ良い」設計は提案しない。
- **AI生成コンテンツの開示**（YouTube の合成コンテンツ開示等）に従う。
- 音楽・画像・動画の生成ツールは**商用利用可のものを選び、商用権を明文で確認**してから使う。
- フォント・効果音・素材は**商用可ライセンスのみ**。ライセンス台帳で管理。
- データ取得は**公式 API のみ**。スクレイピングは ToS 違反のため採用しない。

## ⚠️ 推測で断定しないこと（必ず最新情報を確認）
プラットフォームの API 制限・規約は頻繁に変わる。以下は**実装/着手時に各公式ドキュメントで再確認**し、推測で断定しない：
- YouTube Partner Program の参加条件
- AI 音楽/画像/動画ツールの商用利用権・出力の帰属
- YouTube / Instagram の AI生成開示義務
- Instagram Graph API の Business 要件・自動投稿制約・レート制限
- YouTube Data API / Analytics API の quota

## 作業の進め方

- 提案時は「自動化できるか」「無料で済むか」「規約に触れないか」を必ずチェックする。
- フェーズは `docs/PLAN.md` の G（Phase 0〜3）に従う。現在地を意識して着手する。
